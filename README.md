# Personal Portfolio Website

## Overview
This is a Full Stack Personal Portfolio Website developed to showcase personal information, skills, projects, and contact details. The project demonstrates frontend development, backend development, and basic database integration.

## Features
- Home Page
- About Page
- Projects Page
- Contact Page
- Responsive User Interface
- Backend API using Node.js and Express.js
- Project Data Storage

## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Planned Integration)
- Database Module for Project Data Storage

## Pages

- Provides API endpoints.
