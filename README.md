# Personal Portfolio Website

## About Project
This is a Full Stack Personal Portfolio Website developed using HTML, CSS, JavaScript, Node.js and Express.js.

## Features
- Home Page
- About Page
- Projects Page
- Contact Page
- Backend API using Express.js

## Technologies Used
### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js
